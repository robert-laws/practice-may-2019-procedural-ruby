# Operators

Arithmetic Operators

|Operator|Operation|
|--------|---------|
|+|addition|
|-|subtraction|
|*|multiplication|
|/|division|
|**|exponentiation|
|%|modulus (remainder on division)|

Logical Operators

|Operator|Operation|
|--------|---------|
|&&|logical and|
|\|\||logical or|
|!|not|

Comparison Operators

|Operator|Operation|
|--------|---------|
|==|equal|
|!=|not equal|
|>|greater than|
|<|less than|
|>=|greater than or equal|
|<=|less than or equal|
