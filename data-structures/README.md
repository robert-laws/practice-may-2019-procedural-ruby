# Data Structures

A common data structure is a hash with an array as the values, but can store any data type.

```ruby
team = {
  managers: ["Bob", "Hal", "Jim"],
  players: ["Kal", "Lou", "Mel", "Don", "Abe", "Sam"]
}
```

```ruby
team = {
  Lions: {
    manager: "Bob",
    city: "Denver",
    origin_year: 1999
  },
  Bears: {
    manager: "Gil",
    city: "Santa Fe",
    origin_year: 2012
  }
}
```
